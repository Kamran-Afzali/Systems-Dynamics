Chapter 12 Spirals and Stable Emotional Cycles

Mental health often involves recurrent emotional patterns: recurrent depressive episodes, cycles of anxiety and relief, cycles of social withdrawal and engagement. A natural question is: can these patterns become arbitrarily complex in a “simple” model with only two interacting psychological variables (say, mood and stress), or are there built‑in constraints?

The Poincaré–Bendixson theorem tells us that in a continuous 2D system of ODEs, if trajectories stay bounded and do not settle onto a fixed point, they must approach a periodic orbit (limit cycle). Interpreted psychologically, if we model a person’s mood \(M(t)\) and stress \(S(t)\) as a 2D continuous dynamical system with appropriate regularity, then persistent, bounded, non‑equilibrium dynamics must be rhythmic rather than chaotic. Chronic mood fluctuations in such a model would appear as stable emotional cycles.12

12.0.1 A toy mood–stress system

Consider:

\[ \dot{M} = -M + \tanh(S) + I, \]

\[ \dot{S} = -\alpha S + \beta M, \]

where:

  • \(M(t)\): “mood level” (negative: depressed, positive: elevated).
  • \(S(t)\): “stress/arousal”.
  • \(I\): baseline external input (e.g., social support).
  • \(\alpha > 0\): stress decay rate (self‑regulation).
  • \(\beta\): coupling from mood to stress.34

Under some parameter choices, this system has a stable fixed point (resilient mood). Under others, it can exhibit a stable oscillation (recurrent mood episodes). Poincaré–Bendixson tells us that in this 2D setting, if there is persistent bounded non‑stationary behavior, it must be periodic, not chaotic.

12.0.2 R code: phase portrait and trajectories

library(deSolve)
library(ggplot2)

mood_stress <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dM <- -M + tanh(S) + I
    dS <- -alpha * S + beta * M
    list(c(dM, dS))
  })
}

parms <- list(I = 0.2, alpha = 0.5, beta = 1.2)
times <- seq(0, 100, by = 0.1)

ics <- rbind(
  c(M = -1, S = 0),
  c(M = 0.5, S = 1),
  c(M = 1.5, S = -0.5)
)

traj_list <- lapply(1:nrow(ics), function(i) {
  as.data.frame(ode(y = ics[i, ], times = times,
                    func = mood_stress, parms = parms))
})

traj_df <- do.call(rbind, lapply(seq_along(traj_list), function(i) {
  cbind(traj_list[[i]], id = factor(i))
}))

ggplot(traj_df, aes(x = M, y = S, color = id)) +
  geom_path(alpha = 0.9) +
  coord_equal() +
  labs(title = "Mood–Stress Dynamics",
       x = "Mood M", y = "Stress S") +
  theme_minimal()

ggplot(traj_df, aes(x = time, y = M, color = id)) +
  geom_line() +
  labs(title = "Mood over Time", x = "Time", y = "M(t)") +
  theme_minimal()

Depending on parameters, you can see trajectories converging to a point (stable mood) or to an orbit (recurrent mood cycles). In either case, the theorem guarantees that you do not get sustained chaos in such a 2D continuous model.

12.1 Limit Cycles as Self‑Maintaining Symptom Rhythms

Many mental health conditions show self‑sustained symptom rhythms: circadian mood oscillations in bipolar disorder, binge–restrict cycles in eating disorders, or repeated avoidance–relief cycles in anxiety. These are not neutral oscillations; they persist with characteristic amplitude and period despite disturbances.

In dynamical‑systems terms, these are modeled as limit cycles: isolated closed orbits that are attracting. Nearby trajectories are drawn to the cycle; perturbations fade, and the system returns to its symptomatic rhythm.56

12.1.1 A symptom oscillation model

Let \(X(t)\) represent an aggregate “symptom activation” (e.g., avoidance behavior), and \(R(t)\) a slower “regulatory resource” (e.g., coping capacity):

\[ \dot{X} = X - \frac{X^3}{3} - R + u, \]

\[ \dot{R} = \frac{1}{\tau}(X + a - bR), \]

adapted from FitzHugh–Nagumo.78

  • For some parameters, the system has a stable fixed point (symptoms damp out).
  • For others, it has a stable limit cycle: symptom spikes and partial recoveries repeat autonomously, reminiscent of recurrent compulsive behaviors or cyclical mood disorders.

12.1.2 R code: limit cycle in symptom dynamics

library(deSolve)
library(ggplot2)

symptom_cycle <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dX <- X - X^3/3 - R + u
    dR <- (X + a - b * R) / tau
    list(c(dX, dR))
  })
}

parms <- list(a = 0.7, b = 0.8, tau = 12, u = 0.4)
times <- seq(0, 120, by = 0.05)
y0 <- c(X = 0.5, R = 0)

out <- as.data.frame(ode(y = y0, times = times,
                         func = symptom_cycle, parms = parms))

ggplot(out, aes(x = X, y = R)) +
  geom_path(color = "darkred") +
  coord_equal() +
  labs(title = "Symptom–Regulation Limit Cycle",
       x = "Symptom activation X", y = "Regulation R") +
  theme_minimal()

ggplot(out, aes(x = time, y = X)) +
  geom_line(color = "darkred") +
  labs(title = "Self‑Sustained Symptom Oscillations",
       x = "Time", y = "X(t)") +
  theme_minimal()

This toy system can represent a person whose symptoms repeatedly flare (large \(X\)) and then subside as regulatory resources slowly catch up, only to cycle again—an endogenous rhythm rather than a purely stimulus‑driven response.910

12.2 Floquet Theory and Stability of Treatment Routines

Therapeutic interventions (e.g., weekly CBT sessions, medication cycles, sleep hygiene routines) often aim to stabilize a healthy periodic pattern: regular sleep–wake cycles, exercise schedules, social contact rhythms. Even if a person’s symptom dynamics have an underlying oscillation, the question is whether that rhythm is robust or fragile to perturbations (e.g., stressors, missed sessions).

If we model such a healthy routine as a periodic orbit \(x_0(t)\) of period \(T\), Floquet theory tells us how small deviations evolve. Let \(\delta(t)\) be a small perturbation. Linearizing gives:1112

\[ \dot{\delta} = J(x_0(t))\,\delta, \]

where \(J\) is the Jacobian along the cycle. The nontrivial Floquet exponent \(\mu\) (in 2D) determines whether deviations from the routine:

  • Decay (Re\(\,\mu < 0\)): disturbances fade; the routine is stable.
  • Grow (Re\(\,\mu > 0\)): the routine is structurally fragile.

12.2.1 A basic treatment‑adherence cycle

Let \(A(t)\) be “adherence” to a treatment routine (e.g., doing CBT homework), and \(D(t)\) be “distress level”:

\[ \dot{A} = f(A, D), \quad \dot{D} = g(A, D). \]

We can implement a simplified FHN‑like system that has a stable cycle corresponding to a healthy weekly rhythm of effort and reward.

To illustrate numerically, we can approximate the largest Floquet multiplier for a periodic orbit by tracking how a small perturbation evolves over one period.

12.2.2 R code: approximate Floquet multiplier

library(deSolve)

adherence_system <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dA <- A - A^3/3 - D + u
    dD <- (A + a - b * D) / tau
    list(c(dA, dD))
  })
}

parms <- list(a = 0.5, b = 0.8, tau = 10, u = 0.3)

# 1) Find a long trajectory and estimate period
times <- seq(0, 300, by = 0.05)
y0 <- c(A = 0.2, D = 0)
traj <- as.data.frame(ode(y = y0, times = times,
                          func = adherence_system, parms = parms))

# crude period estimate: detect peaks in A
A <- traj$A
t <- traj$time
peak_idx <- which(diff(sign(diff(A))) == -2) + 1
T_est <- mean(diff(t[peak_idx]))

T_est

# 2) Estimate Floquet multiplier by perturbing initial condition
eps <- 1e-4
y0_plus <- c(A = traj$A[nrow(traj)] + eps,
             D = traj$D[nrow(traj)])

times_short <- seq(0, T_est, by = 0.01)
base_traj <- as.data.frame(ode(y = traj[nrow(traj), c("A","D")],
                               times = times_short,
                               func = adherence_system, parms = parms))
pert_traj <- as.data.frame(ode(y = y0_plus,
                               times = times_short,
                               func = adherence_system, parms = parms))

delta0 <- sqrt((pert_traj$A[^1] - base_traj$A[^1])^2 +
               (pert_traj$D[^1] - base_traj$D[^1])^2)
deltaT <- sqrt((pert_traj$A[nrow(pert_traj)] - base_traj$A[nrow(base_traj)])^2 +
               (pert_traj$D[nrow(pert_traj)] - base_traj$D[nrow(base_traj)])^2)

floquet_multiplier <- deltaT / delta0
floquet_multiplier

If floquet_multiplier < 1, small deviations from the treatment rhythm shrink each cycle: attending therapy late one week doesn’t derail the long‑term routine. If it is > 1, the same routine is unstable and may require redesign.1314

12.3 Hopf Bifurcations and Onset of Recurrent Symptoms

In mental health, a person can shift from a relatively steady baseline (e.g., remitted depression) to recurrent episodes as parameters change: increased stress load, reduced social support, or altered neurobiology (e.g., serotonin or dopamine function).1516

In a 2D system, a common mechanism for such transitions is the Hopf bifurcation. A stable fixed point loses stability when a pair of complex conjugate eigenvalues crosses the imaginary axis, leading to the birth of a limit cycle.1718

12.3.1 Example: stress‑sensitivity parameter

Take again the symptom system

\[ \dot{X} = X - \frac{X^3}{3} - R + u, \]

\[ \dot{R} = \frac{1}{\tau}(X + a - bR), \]

and interpret parameter \(u\) as “chronic stress load” or “cognitive bias towards negative interpretation” (i.e., environmental or cognitive vulnerability). For low \(u\), the equilibrium could be stable (no episodes). As \(u\) increases, the equilibrium may undergo a Hopf bifurcation and produce persistent episodes.

12.3.2 R code: sweep parameter and measure amplitude

library(deSolve)
library(ggplot2)

symptom_system <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dX <- X - X^3/3 - R + u
    dR <- (X + a - b * R) / tau
    list(c(dX, dR))
  })
}

parms_base <- list(a = 0.7, b = 0.8, tau = 12)
u_vals <- seq(0.0, 0.9, by = 0.05)
amps <- numeric(length(u_vals))

times <- seq(0, 300, by = 0.1)

for (i in seq_along(u_vals)) {
  parms <- c(parms_base, list(u = u_vals[i]))
  out <- as.data.frame(ode(y = c(X = 0.1, R = 0),
                           times = times,
                           func = symptom_system, parms = parms))
  tail_seg <- tail(out, 400)
  amps[i] <- max(tail_seg$X) - min(tail_seg$X)
}

bif_df <- data.frame(u = u_vals, amplitude = amps)

ggplot(bif_df, aes(x = u, y = amplitude)) +
  geom_line(color = "blue") +
  geom_point(color = "red") +
  labs(title = "Onset of Recurrent Episodes via Hopf-like Transition",
       x = "Chronic stress / bias (u)",
       y = "Symptom amplitude") +
  theme_minimal()

The plot shows a regime where amplitude is near zero (stable baseline) and a regime where amplitude is positive (recurrent episodes). A smooth onset of oscillation as \(u\) crosses a threshold is analogous to a supercritical Hopf bifurcation: a “soft” transition from remission to recurrent episodes.1920

12.4 Multiple Time Scales and Relaxation Oscillations in Burnout

Burnout and crisis in mental health often involve slow accumulation of stress and fatigue, followed by rapid breakdown (e.g., sudden crash, hospitalization), then gradual recovery. This pattern is naturally modeled by relaxation oscillations in slow–fast systems.2122

Let:

  • \(E(t)\): energy or resilience.
  • \(B(t)\): burnout / exhaustion level.

Consider:

\[ \dot{E} = \frac{1}{\epsilon}(F(E, B)), \quad \dot{B} = G(E, B), \]

with \(0 < \epsilon \ll 1\). Here \(E\) changes fast when pushed beyond thresholds, while \(B\) evolves slowly (months of workload, chronic stress).

A simple FHN‑style burnout oscillator:

\[ \dot{E} = E - \frac{E^3}{3} - B + d, \]

\[ \dot{B} = \epsilon (E + a - bB), \]

with small \(\epsilon\). For most of the time, the system drifts slowly along a quasi‑steady manifold in \((E,B)\), then suddenly jumps when a fold is reached—capturing gradual overload and sudden crash.

12.4.1 R code: slow–fast burnout oscillator

library(deSolve)
library(ggplot2)

burnout_model <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dE <- E - E^3/3 - B + d
    dB <- eps * (E + a - b * B)
    list(c(dE, dB))
  })
}

parms <- list(a = 0.7, b = 0.8, d = 0.3, eps = 0.02)
times <- seq(0, 400, by = 0.1)
y0 <- c(E = -1, B = -1)

out <- as.data.frame(ode(y = y0, times = times,
                         func = burnout_model, parms = parms))

ggplot(out, aes(x = time, y = E)) +
  geom_line(color = "darkred") +
  labs(title = "Relaxation Oscillations in Energy",
       x = "Time", y = "Energy / resilience E") +
  theme_minimal()

ggplot(out, aes(x = E, y = B)) +
  geom_path(color = "darkblue") +
  coord_equal() +
  labs(title = "Burnout Phase Portrait",
       x = "E", y = "B") +
  theme_minimal()

You should see slow drifts and sharp transitions—an abstract analogue of slow buildup to burnout, followed by a rapid crash and then slow recovery.23

12.5 Disease States as Dynamical Regime Shifts in Mental Health

Mental health “disease states” (e.g., major depressive episode, panic disorder) can be conceptualized as regime shifts in an underlying dynamical system: moving from one attractor (healthy equilibrium or mild fluctuations) to another (deep depressive basin, high‑arousal anxiety attractor).2425

In a 2D model, changing parameters can:

  • Create or destroy equilibria (e.g., a new “depressed” equilibrium).
  • Change their stability (e.g., normal state becomes unstable).
  • Alter limit cycles (e.g., from stable circadian rhythm to pathological mood cycles).

12.5.1 Example: two equilibria, healthy and depressed

Consider a mood–rumination model:

\[ \dot{M} = -M^3 + M - R + I, \]

\[ \dot{R} = \gamma (M - R), \]

where \(R\) is rumination. The cubic in \(M\) allows for multiple equilibria: one with positive mood (healthy), one with negative mood (depressed).26

Parameter \(I\) (social support, treatment input) can tilt the system so that the healthy equilibrium is unique and stable, or so that both a healthy and a depressed equilibrium coexist (bistability). Small shocks can then push the system from one basin to another.

12.5.2 R code: two basins (healthy vs depressed)

library(deSolve)
library(ggplot2)

mood_rumination <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dM <- -M^3 + M - R + I
    dR <- gamma * (M - R)
    list(c(dM, dR))
  })
}

parms <- list(I = 0.1, gamma = 0.5)
times <- seq(0, 200, by = 0.1)

ics <- expand.grid(M = seq(-2, 2, length.out = 8),
                   R = seq(-2, 2, length.out = 8))

traj_list <- lapply(1:nrow(ics), function(i) {
  as.data.frame(ode(y = c(M = ics$M[i], R = ics$R[i]),
                    times = times,
                    func = mood_rumination, parms = parms))
})

traj_df <- do.call(rbind, lapply(seq_along(traj_list), function(i) {
  cbind(traj_list[[i]], id = factor(i))
}))

ggplot(traj_df, aes(x = M, y = R, group = id)) +
  geom_path(alpha = 0.6) +
  coord_equal() +
  labs(title = "Healthy vs Depressed Basins in Mood–Rumination Model",
       x = "Mood M", y = "Rumination R") +
  theme_minimal()

You will typically see trajectories ending in different regions: a high‑mood attractor and a low‑mood attractor, depending on initial conditions—consistent with the idea of vulnerability to transitions into, and out of, depressive episodes.2728

12.6 Sensitive Dependence and “Tipping Points” in Mental Health

Even if 2D continuous systems cannot exhibit sustained chaos, they can have regions of strong sensitivity near saddles or separatrices. In mental health, this corresponds to tipping points where small differences (e.g., one extra stressor, a slightly different cognitive appraisal) can have large long‑term consequences (relapse vs recovery).2930

Near a saddle point, trajectories diverge exponentially along the unstable direction. The largest Lyapunov exponent \(\lambda\) near such a point can be approximated from the Jacobian eigenvalues.

12.6.1 Simple tipping‑point system

Take again the mood–rumination system:

\[ \dot{M} = -M^3 + M - R + I, \]

\[ \dot{R} = \gamma (M - R). \]

For certain \(I\), the system has a saddle equilibrium separating healthy and depressed basins. Close initial conditions on opposite sides of the saddle’s stable manifold eventually diverge to very different outcomes.

12.6.2 R code: divergence of nearby states

library(deSolve)
library(ggplot2)

mood_rumination <- function(t, state, parms) {
  with(as.list(c(state, parms)), {
    dM <- -M^3 + M - R + I
    dR <- gamma * (M - R)
    list(c(dM, dR))
  })
}

parms <- list(I = 0.0, gamma = 0.5)
times <- seq(0, 80, by = 0.05)

y0_1 <- c(M = -0.05, R = 0.0)
y0_2 <- c(M =  0.05, R = 0.0)

traj1 <- as.data.frame(ode(y = y0_1, times = times,
                           func = mood_rumination, parms = parms))
traj2 <- as.data.frame(ode(y = y0_2, times = times,
                           func = mood_rumination, parms = parms))

sep <- sqrt((traj1$M - traj2$M)^2 + (traj1$R - traj2$R)^2)
sep_df <- data.frame(time = times, sep = sep)

ggplot(sep_df, aes(x = time, y = log(sep))) +
  geom_line(color = "darkblue") +
  labs(title = "Divergence of Nearby Mental States near a Tipping Point",
       x = "Time", y = "log(separation)") +
  theme_minimal()

Initially small differences in mood around 0 can lead to trajectories diverging toward very different long‑term states (recovery vs chronic depression), capturing the notion that early, small interventions near a tipping point can have disproportionate impact.3132

12.7 The Edge of Chaos and Complexity in Mental Health Trajectories

While 2D continuous systems cannot sustain full‑blown chaos, they can sit close to bifurcation points or homoclinic structures where dynamics become very complex: long transients, high sensitivity, and multiple timescales. In mental health, this corresponds to states where individuals are highly vulnerable: small changes in environment, treatment adherence, or self‑beliefs might lead to very different paths.3334

In higher dimensions (adding, e.g., separate cognitive, emotional, and social variables), chaos becomes fully possible, and computational psychiatry increasingly considers such models.35

A simple 2D system cannot fully capture this, but it can approximate the idea of “edge of stability” where dynamics are slow, sensitive, and structurally rich near bifurcations.
